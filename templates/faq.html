📞 Phone & Calling

I called the agent but it said "application error" and cut the call Fixed ▼
Your Twilio phone number isn't assigned to an ElevenLabs agent yet. Go to elevenlabs.io → Conversational AI → Phone Numbers → click your number → assign the Vendor Research Coordinator agent. After that, calls will connect properly.
It plays "You have a trial account" every time I call Info ▼
This is Twilio's trial account restriction. You can't skip it without upgrading. Go to twilio.com/console → Billing → Upgrade Account and add a payment method. Your $15 free credit carries over — upgrading just removes the trial message.
Outbound calls to India keep getting cut immediately Fixed ▼
Twilio trial accounts block international calls entirely, even if you have credit. You must upgrade your Twilio account to a paid account first. International calls (US → India) cost roughly $0.05–0.08/min. Your $15 credit is more than enough for testing.
I can't get an Indian (+91) phone number on Twilio Info ▼
Twilio requires TRAI regulatory documents for Indian numbers — it's an Indian government rule, not Twilio's choice. Your options:
  • Use a US Twilio number — works fine for calling Indian vendors, just shows a US caller ID
  • Plivo — Indian company with +91 numbers, ElevenLabs supports it natively
  • Exotel — popular Indian provider, connects via SIP trunking to ElevenLabs
For production with Indian clients, Exotel is the go-to choice.

🤖 Telegram Bot

My Telegram bot isn't responding to /start Fixed ▼
Two things to check:
  • Make sure the server is running: python3 -m uvicorn main:app --host 0.0.0.0 --port 8000
  • Make sure your tunnel is active (cloudflared or ngrok) and SERVER_URL is set correctly in .env
Telegram delivers messages to your /telegram/webhook endpoint. If the server isn't running or the URL is wrong, messages silently fail.
Telegram webhook shows "501 Not Implemented" error Fixed ▼
Another process was running on port 8000 intercepting requests. Run lsof -i :8000 to see all processes on that port, then kill any that aren't your uvicorn server. Then restart uvicorn with --host 0.0.0.0 so it listens on all interfaces.
How do I link my phone number to receive Telegram notifications? Info ▼
Open your Telegram bot and send: /link +919010118787 (replace with your number, include country code). This must be the same number you call the agent from. The system uses this to match your inbound call to your Telegram account for live updates.

🔑 API Keys & Setup

What ElevenLabs API key permissions do I need? Info ▼
Only 4 permissions needed — set everything else to No Access:
  • ElevenAgents — Write
  • Webhooks — Access
  • Voices — Read
  • User — Read
Server crashes on startup with "unsupported operand type(s) for |: 'type' and 'NoneType'" Fixed ▼
This happens on Python 3.9 — the str | None union type syntax requires Python 3.10+. Either upgrade to Python 3.10+ or replace str | None with Optional[str] from the typing module in the Pydantic models.
How do I expose my local server to the internet for webhooks? Info ▼
Use cloudflared (free, no signup needed):

cloudflared tunnel --url http://localhost:8000 --no-autoupdate

It gives you a URL like https://xxx.trycloudflare.com — paste that as your SERVER_URL in Settings. Note: this URL changes every time you restart the tunnel, so you'll need to re-run setup_agents.py after each restart in development.

🔍 Vendor Research

How does the system find vendor phone numbers? Info ▼
It follows this priority order:
  • 1. Your vendor database — if you've added the vendor via python manage_vendors.py add, it uses that data instantly
  • 2. Firecrawl web search — searches the vendor's website, scrapes it for phone number, contact info, and any listed pricing
  • 3. Internet search for alternatives — if a vendor can't fulfill, you'll get a Telegram prompt asking if you want to search for other suppliers
How do I add my existing vendors to the database? Info ▼
Run python manage_vendors.py add and fill in the prompts: vendor name, phone, website, what they supply, min/max order quantities, and any notes. To see all vendors: python manage_vendors.py list. To remove one: python manage_vendors.py remove "Vendor Name".
