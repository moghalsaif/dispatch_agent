Settings

Configure your API keys. All keys are stored locally in your database and never shared.

🤖 ElevenLabs

Get it from elevenlabs.io → Settings → API Keys. Needs ElevenAgents Write + Webhooks Access permissions.

From Conversational AI → Phone Numbers after connecting Twilio.

🔍 Firecrawl

Get it from firecrawl.dev → Dashboard → API Keys. Used to search vendor websites and scrape phone numbers + pricing.

📱 Telegram Bot

Create a bot via @BotFather on Telegram → /newbot → copy the token.

🌐 Server

Your public URL for webhooks. Use cloudflared tunnel --url http://localhost:8000 for local dev, or your production server URL.

🚀 Setup Steps

1
Fill in all API keys above and click Save Settings
2
Run python setup_agents.py — this creates your ElevenLabs inbound + outbound agents automatically
3
In ElevenLabs dashboard → Conversational AI → Phone Numbers → assign the Vendor Research Coordinator agent to your number
4
Open your Telegram bot → send /link +your_phone_number (same number you'll call from)
5
Add your existing vendors: run python manage_vendors.py add
6
Call your Twilio number → tell the agent which vendors to contact, what product, and quantity → done!
